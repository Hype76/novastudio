# Nova Studio

A SaaS foundation for building projects.

## Setup

1. Copy `.env.example` to `.env.local` (create it if missing)
2. Add your Supabase URL and Anon Key
3. Run `npm install`
4. Run `npm run dev`

## Deployment

For Netlify, copy `netlify-deploy-config.txt` to `netlify.toml` in the root directory before pushing to GitHub.
